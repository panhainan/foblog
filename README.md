# fo-blog
响应式开源个人博客（简洁清爽风格）
# 使用手册

注意：

作者管理账户的登录地址为：http://ip:port(/project)/#!/manage/sign
这里需要到JUnit测试类
```studio.baxia.fo.service.IUserServiceTest```
中运行`testAddAuthors`测试方法，根据自身情况设置用户名密码，然后运行即可。
这样就可以通过刚刚生成的用户名密码进行登录了。
